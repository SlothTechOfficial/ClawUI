# ClawUI 

This is a UI component template for an event website, it is open source and you can modify it for your events if you need it,


## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files

```
